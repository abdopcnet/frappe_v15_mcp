# field_properties

Hide, enable, require, or set options on a field.

```javascript
frm.toggle_display("field_name", true);
frm.toggle_enable("field_name", true);
frm.toggle_reqd("field_name", true);
frm.set_df_property("field_name", "options", "A\nB\nC");
frm.set_df_property("field_name", "hidden", 1);
frm.set_df_property("field_name", "read_only", 1);
frm.set_df_property("field_name", "reqd", 1);
```
