# Skills README

This folder is a snippet-first reference for Frappe and ERPNext work.

## How to use it

- Open the smallest file that matches the task.
- Prefer files with a single question and 2 to 4 examples.
- Follow boundaries between similar topics instead of merging concepts.

## Category shortcuts

- `api/`: whitelisted methods and HTTP exposure
- `backend/`: controller lifecycle, save, validation
- `database/`: read, write, query, transaction patterns
- `frontend/`: Desk form and list behavior
- `system/`: hooks, jobs, cache, permissions, scheduler
- `bench/`: CLI and site operations

## Style target

- Short docs
- Snippet-first
- Real Frappe patterns
- Minimal duplication
