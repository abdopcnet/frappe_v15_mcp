# ERPNext / Frappe MCP Context — v15

Use this folder as Frappe/ERPNext v15 reference in Cursor or VSCode.

## GitMCP (remote)

- Push this repo to GitHub, then in Cursor/VSCode add MCP URL: `https://gitmcp.io/username/repo`
- Replace `github.com` with `gitmcp.io` in the repo URL.

## Local (Cursor)

- Add this folder to your project or use the `*.md` files as references.
- Or paste `llms.txt` into Agent instructions for the full index.

## Structure

- `rules/` — Development rules (from AGENTS.md).
- `skills/` — One short instruction per file (Frontend, Backend, DB, API, Bench, etc.).
